## Descripción del proyecto

El sistema permite administrar materiales de una biblioteca
utilizando herencia en Java.

Se implementaron diferentes tipos de materiales:

- Libro
- Revista
- Tesis
- Audiolibro

El sistema permite:

- Agregar materiales
- Listar materiales
- Buscar materiales por ID
- Prestar materiales
- Devolver materiales
- Mostrar total de materiales registrados

## Tecnologías utilizadas

- Java
- NetBeans
- GitHub

## Ejecución

1. Abrir el proyecto en NetBeans
2. Ejecutar la clase Main.java
3. Utilizar el menú en consola
